# GRam_OS


GRam_OS is a simple operating system for the ternary processor **5500FP** and its development system **GargantuRAM**. It is written entirely in ternary Assembly and serves as a starting point for anyone who wants to explore, experiment with, and improve an OS on a ternary architecture.

The project has a dual purpose: to be a concrete example of how to use the 5500FP ternary CPU, and to serve as a proof of its operation.

## Features

GRam_OS provides an interactive shell with the following built-in commands: `cls` (clears the screen), `version` (displays the OS version), `ls` (lists the contents of the SD Card via a simple dedicated filesystem), `run` (launches a program stored on the SD Card).

## Who is it for

This repository is intended for anyone who wants to:

- have a starting point for developing an OS on ternary hardware
- study and understand how a ternary CPU works through real code
- contribute improvements, new commands, or a more advanced filesystem
